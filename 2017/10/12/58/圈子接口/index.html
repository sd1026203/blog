{ "data": null,

   "baseResponse": {

   }
}

Data为具体返回值, 下面会罗列各个接口的返回data baseResponse为状态码
只有status为0的是正常返回, 1为系统异常, 3为参数异常, 4为业务异常, 5猎人审核异常

圈子API

圈子列表 - 分页查询

请求方法: CircleService.getCircles(QueryCirclesModel queryCirclesModel)

参数queryCirclesModel:

{
    "businessID": null, //业务线ID
    "circleID": null,   //圈子ID, 用于分页查询, 查询时输入上次查询的最后一个ID, 首次查询传null
    "pageSize": null    //分页大小
}

圈子列表 - 根据圈子ID查询

请求方法: CircleService.getCircles(APPQueryCirclesByRowKeyAndColKeysModel appQueryCirclesByRowKeyAndColKeysModel)

参数APPQueryCirclesByRowKeyAndColKeysModel:
{
    "businessID": null, //业务线ID
    "circleIDs": []     //圈子ID集合
}


返回值:List
APPCircleModel格式为:
{
    "title": null,        //圈子名称
    "description": null,  //圈子描述
    "pic": null,          //圈子图片
    "businessID": null,   //业务线ID
    "circleID": null      //圈子ID
}

关注圈子

请求方法: CircleService.followCircle(APPFollowCircleModel appFollowCircleModel)
参数appFollowCircleModel:

{
    "circleID": null,   //圈子ID
    "title": null,      //圈子名称
    "user": null,       //关注用户, 为User对象
    "businessID": null  //业务线ID
}

返回值: Boolean

关注圈子列表

请求方法: CircleService.whoFollowsCircle(APPWhoFollowsCircleModel appWhoFollowsCircleModel);
参数appWhoFollowsCircleModel:

{
    "circleID": null,    //圈子ID
    "businessID": null,  //业务线ID
    "pageNum": null,     //第几页
    "pageSize": null     //每页几条数据
}

返回值: WhoFollowsCircleModel

{
 "totalCount":null,
 "whoFollows":[]
}

主题API

发表主题

请求方法: TopicService.APPAddTopicModel(APPAddTopicModel appAddTopicModel)

参数appAddTopicModel

{
    "content":null,     //内容
    "imgs":null,        //图片
    "videos":null,      //视频
    "user":null,        //发表用户
    "topicID":null,     //动态ID, 新动态传null
    "businessID":null,  //业务线ID
    "circleIDs":null,    //圈子ID列表
    "cities":null,      //所属城市
    "source":null       //来源, 1:用户, 2:编辑
}

返回值:TopicModelResponse
{
    "content":null,     //内容
    "imgs":null,        //图片
    "videos":null,      //视频
    "user":null,        //发表用户
    "createdTime":null, //创建时间
    "modifiedTime":null,//修改时间
    "essencedTime":null,//加精时间
    "isdel":null,       //是否被删除, 0:未删除, 1:删除
    "checkStatus":null, //审核状态
    "topicID":null,     //动态ID
    "businessID":null,  //业务线ID
    "circleIDs":null,    //圈子ID列表
    "rejectReason":null,//驳回原因
    "cities":null,      //城市
    "source":null       //来源
}
